# 自动记账 APP · 三主题全量 UI 原型

这是一个与 Android 业务源码隔离的高保真交互原型，用于比较并选定下一版 APP 的完整视觉方向。

## 预览

在本目录启动任意静态 HTTP 服务，例如：

```powershell
python -m http.server 8765 --bind 127.0.0.1
```

然后打开：`http://127.0.0.1:8765/`

也可以直接打开 `index.html`，但推荐使用本地 HTTP 服务，以保证浏览器行为一致。

## 三套方案

1. **青芽日常（推荐）**：最贴近现有使用说明的绿色、圆润和插画气质。
2. **阳光票据**：米黄色纸张感、票据虚线和更温暖的生活记录氛围。
3. **薄荷数据**：更紧凑清晰的数据布局，适合统计、限额和诊断场景。

三套方案共用页面结构，点击左侧方案名称即可即时切换，不会丢失当前页面。

## 页面覆盖

- 首次使用引导
- 记账首页
- 待确认账单
- 新增/编辑账单
- 账单详情
- 截图识别
- 统计分析
- 每日限额
- 自动监听
- 全部工具
- 分类规则与学习记录
- 备份与恢复
- 自动记账故障诊断
- 设置、监控应用与后台运行

页面还包含监听正常、监听中断、权限提示、待确认、识别成功、备份成功等关键状态。

## 已验证

- Chrome/Codex 内置浏览器加载无控制台错误。
- 14 个代表页面均无横向溢出。
- 390px、768px 和桌面宽度响应式布局正常。
- 底部导航固定在手机画板内部。
- 关键路径可点击：首次引导 → 自动监听 → 首页 → 新增账单 → 账单详情。

## 边界

该原型仅用于 UI/UX 评审，不连接 Room 数据库、不读取通知、不执行备份恢复，也不修改任何真实账单。选定方案后，再把组件、设计令牌和页面结构迁移到 Jetpack Compose。
