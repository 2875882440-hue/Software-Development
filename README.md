# 本地自动记账

版本：V1.0.0 Personal Stable

本地自动记账是一款纯本地运行的 Android 自动记账 APP，用于个人日常收支记录、微信/支付宝付款通知监听、账单统计、每日限额提醒以及本地备份恢复。

## 定位

- 纯本地自动记账 APP
- 不联网
- 不登录
- 不上传服务器
- 不使用云同步
- 账单数据保存在手机本地
- 适合个人长期稳定使用

## 已完成功能

- 本地手动记账
- 微信/支付宝付款通知监听
- 自动生成账单
- 待确认账单
- 无金额通知待补录
- 账单编辑与删除
- 重复记录过滤
- 自动分类
- 自定义分类规则
- 日 / 周 / 月统计
- 限额提醒
- 今日不再提醒
- 自动监听状态检测
- 前台服务保活
- 监听急救 / probe 检测
- 手势返回
- CSV 导出
- JSON 完整备份
- JSON 恢复并跳过重复账单

## 使用说明

首次使用建议按顺序完成：

1. 打开 APP。
2. 进入底部“监听”页。
3. 开启通知监听权限。
4. 开启自动监听。
5. 按手机系统要求设置后台权限。
6. 返回“记账”页查看自动生成和待确认账单。

详细操作请阅读 `USER_GUIDE.md`。

## 数据与隐私

- 账单数据仅保存在手机本地数据库中。
- CSV / JSON 导出文件由用户主动选择保存位置。
- JSON 恢复会跳过重复账单。
- APP 不上传账单、通知内容或个人信息到服务器。
