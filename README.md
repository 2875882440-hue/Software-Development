# 芽芽记账

版本：V1.2.1 稳定化维护版

芽芽记账是一款纯本地运行的 Android 自动记账 APP，用于个人日常收支记录、微信/支付宝付款通知监听、账单统计、每日限额提醒以及本地备份恢复。

## 定位

- 纯本地运行的自动记账 APP
- 不联网
- 不登录
- 不上传服务器
- 不使用云同步
- 账单数据保存在手机本地
- 适合个人长期稳定使用

## 已完成功能

- 本地手动记账
- 微信/支付宝付款通知监听
- 自动生成账单
- 待确认账单
- 无金额通知待补录
- 账单编辑与删除
- 重复记录过滤
- 自动分类
- 自定义分类规则
- 日 / 周 / 月统计
- 限额提醒
- 今日不再提醒
- 自动监听状态检测
- 前台服务保活
- 监听急救 / probe 检测
- 手势返回
- CSV 导出
- JSON 完整备份
- JSON 恢复并跳过重复账单
- 可选择微信、支付宝及其他已安装应用参与通知识别
- 最近 30 天本地匿名自动记账计数
- 最近 14 天微信 / 支付宝 / 其他应用成功率报告
- 脱敏问题日志和后台诊断报告

## 使用说明

首次使用建议按顺序完成：

1. 打开 APP。
2. 进入底部“监听”页。
3. 开启通知监听权限。
4. 开启自动监听。
5. 按手机系统要求设置后台权限。
6. 返回“记账”页查看自动生成和待确认账单。

详细操作请阅读 `USER_GUIDE.md`。

## 数据与隐私

- 账单数据仅保存在手机本地数据库中。
- CSV / JSON 导出文件由用户主动选择保存位置。
- JSON 恢复会跳过重复账单。
- APP 不上传账单、通知内容或个人信息到服务器。
- 自动记账成功率只保存日期、来源分组、事件类型和计数，不保存商户、金额、通知正文或账单内容。

## 当前状态

V1.2.1 是基于 V1.2.0 UI Refresh 的稳定化维护版，重点统一项目文档、加入匿名自动记账成功率统计、建立多机型测试流程并生成可校验安装包。

当前仍以个人、小范围测试使用为主。V1.2.2 将重点进行小米 / Redmi、华为 / 荣耀、OPPO、vivo、三星及不同 Android 版本的长期监听验证。

维护边界：

- 不大规模重写 UI。
- 不随意修改 Room schema 12。
- 不破坏通知监听、支付解析、限额和备份恢复核心逻辑。
- 不加入账号、云同步、会员或服务器依赖。
