# USER_GUIDE

## 1. 如何开启通知监听权限

1. 打开“芽芽记账”。
2. 进入底部“监听”页。
3. 点击“打开通知监听权限”或系统提示中的权限入口。
4. 在系统“通知使用权 / 通知监听”页面找到本 APP。
5. 开启授权后返回 APP。
6. 在“监听”页确认“通知监听权限”显示为“已授权”。

## 2. 如何开启自动监听

1. 进入“监听”页。
2. 点击“开启自动监听”。
3. 阅读提示并确认。
4. 系统状态栏会出现前台服务通知，用于提高后台监听稳定性。
5. 如需停止，回到“监听”页点击“关闭自动监听”。

## 3. 小米 / 红米后台权限设置建议

小米、红米、MIUI / HyperOS 对后台服务限制较严格。建议手动检查：

- 应用信息 -> 省电策略 -> 无限制
- 应用信息 -> 自启动 -> 允许
- 应用信息 -> 通知管理 -> 允许通知
- 微信通知管理 -> 允许通知
- 支付宝通知管理 -> 允许通知
- 锁屏通知 -> 显示通知内容
- 横幅通知 -> 开启
- 后台弹出界面 -> 允许
- 最近任务列表中锁定本 APP

## 4. 如何使用记账功能

1. 进入“记账”页。
2. 点击“记一笔”。
3. 输入金额。
4. 选择“支出”或“收入”。
5. 选择分类。
6. 可选填写商户和备注。
7. 点击“保存”。

## 5. 如何确认待确认账单

1. 进入“记账”页。
2. 点击“待确认”入口。
3. 检查自动识别出的金额、分类、商户和时间。
4. 信息正确时点击确认。
5. 信息不完整或不准确时，先编辑再确认。

## 6. 如何编辑 / 删除账单

1. 在“记账”页打开最近账单或统计分组中的账单。
2. 进入账单详情。
3. 点击“编辑”修改金额、类型、分类、商户、备注或时间。
4. 点击“删除”可删除账单。

## 7. 如何设置每日限额

1. 进入底部“限额”页。
2. 输入每日限额金额，例如 `100`。
3. 点击“设置每日限额”。
4. 页面会显示“今日已消费”和“剩余额度”。
5. 超额时会提示“已超额”。
6. 如当天不想再收到提醒，可在提醒弹窗中选择“今日不再提醒”。
7. 如不再使用限额，点击“关闭限额”。

## 8. 如何查看日 / 周 / 月统计

1. 进入“记账”页。
2. 点击“统计”。
3. 可查看今日、本周、本月或自定义范围统计。
4. 可按天、按周、按月分组查看账单。
5. 统计页面会展示总收入、总支出、结余、最大支出和最近账单。

## 9. 如何导出 CSV

1. 进入“记账”页。
2. 点击“备份与恢复”。
3. 点击“导出 CSV”。
4. 在系统文件选择器中选择保存位置。
5. CSV 用于表格查看和个人归档。

## 10. 如何导出 JSON 备份

1. 进入“备份与恢复”页。
2. 点击“导出 JSON 备份”。
3. 选择保存位置。
4. JSON 备份包含完整账单数据，适合恢复使用。

## 11. 如何从 JSON 恢复

1. 进入“备份与恢复”页。
2. 点击“从 JSON 恢复”。
3. 选择之前导出的 JSON 文件。
4. APP 会导入账单并跳过重复账单。
5. 恢复完成后查看提示中的导入数量和跳过重复数量。

## 12. 监听失效怎么办

如果发现付款没有自动记录：

1. 进入“监听”页。
2. 查看“通知监听权限”是否仍为“已授权”。
3. 查看“自动监听”是否为“已启用”。
4. 查看“前台服务”是否为“运行中”。
5. 如果异常，点击“一键修复”或“运行诊断修复”。
6. 如果仍未恢复，重新进入系统通知监听权限页面关闭再开启授权。
7. 检查微信/支付宝是否真的发送了付款通知。

## 13. 如何使用监听急救

1. 进入“监听”页。
2. 点击“运行诊断修复”。
3. APP 会检查通知监听权限、前台服务、监听连接和 probe 检测。
4. 根据页面结果处理异常项。
5. 如果显示监听仍未恢复，请检查系统权限并稍后重试。

## 14. 如何避免忘记开启自动监听

- 每次重启手机后打开 APP 看一眼“监听”页。
- 确认“自动监听”为“已启用”。
- 确认“前台服务”为“运行中”。
- 保持 APP 通知权限开启。
- 在小米 / 红米手机上设置自启动和省电策略为无限制。
